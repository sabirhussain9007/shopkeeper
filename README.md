# Shopkeeper Management System

Production-grade Next.js 16 retail management system foundation.

## Phase 1 Included

- Next.js 16 App Router with TypeScript, Tailwind CSS, strict config, ESLint, and Prettier.
- MongoDB/Mongoose models for users, roles, products, categories, customers, suppliers, sales, sale items, purchases, purchase items, ledger entries, settings, and stock adjustments.
- Auth.js/NextAuth credentials auth, JWT sessions, protected routes, and role based permissions for Admin, Manager, and Cashier.
- Route Handler CRUD APIs with validation, soft deletes, audit fields, pagination, search, and permissions.
- POS checkout API with stock validation, credit limit enforcement, stock decrement, sale item persistence, and ledger entry creation.
- Dashboard, inventory, category, customer, supplier, POS, ledger, sales, purchases, reports, and settings routes.
- Zustand POS store with localStorage persistence for refresh recovery, held orders, cart management, discounts, payment modes, and change calculation.
- Receipt component optimized for 58mm, 80mm, A4, browser print dialog, and thermal printer friendly monochrome output.

## Setup

1. Copy `.env.example` to `.env.local` and fill MongoDB/Auth values.
2. Run `npm install`.
3. Run `npm run seed` to create the default admin, roles, and settings.
4. Run `npm run dev`.

## Seeding

`npm run seed` is safe to run more than once. It creates or updates the default admin from `DEFAULT_ADMIN_EMAIL`, creates the default roles, and creates or backfills business settings.

- Set `SEED_RESET_ADMIN_PASSWORD=true` to reset the default admin password from `DEFAULT_ADMIN_PASSWORD`.
- Set `SEED_OVERWRITE_SETTINGS=true` to replace existing settings with default seed values.

## Production Notes

- Use MongoDB Atlas indexes generated by the Mongoose schemas.
- Set `NEXTAUTH_SECRET` to a strong random secret.
- Configure Vercel environment variables for `MONGODB_URI`, `NEXTAUTH_URL`, and `NEXTAUTH_SECRET`.
- Connect USB, Bluetooth, or network thermal printers through the browser print dialog and select the matching 58mm/80mm paper size.
